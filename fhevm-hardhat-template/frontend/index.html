🔒 Voter (Chiffrement FHE réel)

Processus :
  1. Le SDK FHE génère un ciphertext (euint32) localement
  2. Une preuve (inputProof) est créée via le relayer mock
  3. La transaction est envoyée via ethers.js au contrat
  4. Le contrat additionne le vote chiffré sans le déchiffrer
